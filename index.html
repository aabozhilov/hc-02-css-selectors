1. Type Selector
    1. Define a CSS rule to style all paragraph elements to have blue text and bold font weight.
    2. Define a type selector to style all heading elements to have green text.
    3. Style different HTML elements (h1, p, div) using type selectors.
  2. Class Selector
    1. Define a CSS rule to style all paragraph elements to have blue text and bold font weight.
    2. Write an example CSS rule using a class selector to change the background color of elements with a specific class.
    3. Apply a common style to multiple elements using a class selector.
    4. Change the background color of elements with a specific class.
  3. Percentage Width
    • Set the width of a div to 50% and observe how it changes with the browser window size.
  4. Nested Divs
    • Create a parent div with 60% width and a child div with 50% width of its parent. Apply some colors to better see the difference.
  5. Pixel Width
    • Set the width of a div to 300px and observe its behavior when resizing the browser.
  6. Pixel vs Percentage Width
    • Create two divs, one with a width set in pixels and the other in percentages.
  7. EM Units
    1. Set the font-size of a parent div to 20px and a child div to 1.5em.
    2. Change the parent div's font-size to 10px and observe the change in the child div's font-size.
  8. REM Units
    1. Set the root element's font-size to 16px and a div element's font-size to 2rem.
    2. Change the root element's font-size to 20px and observe the change in the div element's font-size.
  9. Specificity
    1. Use a class and a type selector on the same element and see which style is applied.
    2. Apply both a class and an ID selector to the same element and see which style takes precedence.
  10. ID Selector
    • Style elements with unique IDs.
  11. Attribute Selector
    1. Style input elements based on their type attribute.
    2. Change the text color of links with a specific href attribute.
  12. Universal Selector
    1. Use the universal selector to set a default margin and padding for all elements.
    2. Use the universal selector to set a default font-family and color for all elements.
  13. Descendant Combinator
    • Style nested elements using a descendant combinator.
  14. Combinator Comparison

  15. Selector List

  16. Child Combinator
    1. Use a child combinator to style direct children of a div element.
    2. Use a child combinator to style direct children of a specific class.
  17. Box Model
    1. Use CSS to demonstrate the box model by setting margin, border, padding, and content area values.

  18. Display Property
